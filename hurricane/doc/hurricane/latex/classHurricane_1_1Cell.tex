Multiple instances of a same model are modificated so that each instance has it's own {\itshape unique} copy of the model.

This is equivalent to a flatten operation, but without destroying the intermediate level of hierarchy. Only the {\bfseries netlist} and instance's placements gets duplicated. {\itshape No physical components will be duplicated.}
