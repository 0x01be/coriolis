\hypertarget{classHurricane_1_1Exception_secExceptionIntro}{}\subsection{Introduction}\label{classHurricane_1_1Exception_secExceptionIntro}
The \hyperlink{classHurricane_1_1Exception}{Exception} class groups all exceptions thrown by functions from the A\-P\-I. This virtual class is only useful to catch exceptions originating from one of those functions.

\begin{DoxyParagraph}{Remark\-:}
Copy construction is disabled.
\end{DoxyParagraph}

   \textcolor{comment}{// do something}

   \textcolor{comment}{// Go through here if the exception comes from a function of the API}
